## 作品简介（100字以内）

本作品是一个面向低空经济场景的 Web 端无人机调度可视化平台。系统基于 WebGL 实现三维城市地图渲染，支持 500+ 架无人机同屏飞行动画；后端采用 A* 算法自动规划避障航线；集成通义千问大模型进行飞行前风险预审。同时提供移动端 H5 下单、数据分析看板和基建 ROI 沙盘等功能。

## 创新描述（100字以内）

将大语言模型引入无人机航线安全评审，实现基于 Prompt 的多维风险评估与红/黄/绿灯判定；自研二进制轨迹传输协议（对比 JSON 传输体积压缩 75%、解码提速 40 倍）；基于经纬度格网分桶构建禁飞区空间索引，支持 A* 搜索中的高频碰撞检测。

## 特别说明

1. 地图来源：本作品使用 CARTO Positron 开源底图（基于 OpenStreetMap），不涉及国界行政区划标注，仅用于学术研究展示。页面底部已标注地图合规声明。

2. 前期基础：本作品为参赛团队从零开发，无前期基础。开发周期为 2025 年 12 月至 2026 年 4 月。

3. AI 工具使用：本作品在开发过程中使用了通义千问和 DeepSeek 辅助代码编写与调试，已填写《AI 工具使用说明》。作品运行时调用通义千问 API 实现航线安全预审功能。
