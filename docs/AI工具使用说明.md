## 附录：佐证材料

### 序号 1 的佐证材料

AI 安全预审功能的完整代码实现位于 `backend/api/ai.py`。

System Prompt 和 User Prompt 的构造逻辑见该文件第 88-110 行。模型调用使用阿里云百炼平台的 HTTP 接口（`https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions`），代码中配置了 temperature=0.2 和 response_format=json_object 参数。

前端 AI 预审弹窗组件位于 `frontend/src/components/AiPreflightModal.tsx`，负责向用户展示评估结果。

### 序号 2 的佐证材料

以 `frontend/src/hooks/useUAVAnimation.ts` 为例，该文件共约 900 行代码。初始版本中约 100 行基础框架参考了 DeepSeek 的建议，后续的性能优化（TypedArray 管理、能耗模型集成、告警检测回路、轨迹预编译等）共约 800 行均为团队独立编写。

### 序号 3 的佐证材料

文档润色主要用于 README.md 的功能描述段落。润色前后的差异为措辞调整（如将「实现了一个基于 WebGL 的可视化平台」改为「用 WebGL 做三维渲染」），核心技术参数和实现描述未做改动。
